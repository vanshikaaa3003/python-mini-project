# 🔐 Simple Password Generator (Python)

A lightweight Python program that generates random passwords using letters, numbers, and special characters.

---

## 🚀 Features

* Generates random passwords instantly
* Uses:

  * Uppercase and lowercase letters
  * Digits
  * Special characters
* User-defined password length
* Simple and easy to use

---

## 🛠️ Technologies Used

* Python 3
* Built-in modules:

  * `random`
  * `string`

---

## 📂 Project Structure

```
password-generator/
│── main.py
│── README.md
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/password-generator.git
```

2. Navigate to the folder:

```
cd password-generator
```

3. Run the program:

```
python main.py
```

---

## 🧑‍💻 Usage

* Enter the desired password length
* The program will generate a random password instantly

---

## 💡 Example

```
Enter password length: 10
Generated Password: A@9k#P2!xL
```

---

## 📌 Notes

* Password strength depends on the chosen length
* Recommended length: 8 or more characters
